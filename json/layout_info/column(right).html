Currently the individual video wall displays are configured for 1920 x 1080 resolution (16:9 aspect ratio; 1080p). The overall resolution of the Video wall is (3840 x 2160).

  • All elements are NOT equally sized: 3 total elements.
  • First and second elements fill an entire display: (1920 x 1080).
  • Third element is a column that spans two displays vertically: (1920 x 2160).
  • Aspect ratio required for images/video: 16:9 (first and second) and 8:9 (third).
  • Best display performance is achieved with 1920 x 1080 (first and second) and 1920 x 2160 (third).

The larger the file, the slower the load time for both images and videos. If possible, we recommend the webp format for images, and webm for videos. But other formats that are also optimized for online deliverey are out there.
