JSON Generator Instructions

Slide Definitions

  • Title: This can be any text, no one will see it but it helps you identify your slides.
  • Head: The <head></head> properties added to the page. Custom CSS (you can download the default.css using the button at the top of the screen) could be specfied here.
  • Layout: Specifies the slide layout you want to use for your slide.
  • Layout Customization: application slides require customizations. Leave this as null for general slides.
  • Content Child: specifies the server subfolder for the content files (images/videos) used on the slide.
  • Duration: full duration for the slide to be displayed. You can enter seconds or milliseconds.
  • Fade: The is fade-in time in milliseconds for a slide when it initially loads.
  • Timer: Enable to display a timer on your slide: useful for troubleshooting.
  • Timer Style: Available with a green or red background. More can be added.
  • Begin Rotation: You can specify a time, or date and time for a slide to enter the slideshow.
  • End Rotation: You can specify a time, or date and time for a slide to be removed from the slideshow.
  • Launch: You can specify a time (runs every dat at that time), or date and time for a scheduled slide.
  • Launch Wait: For slides with a scheduled start, how long before do you want the announcement displayed.
  • Launch Wait Slide: Choose the slide used to announce a presentation is beginning.
  • Launch Wait Class: Styles applied to the slide used to announce a presentation is scheduled to begin.

Elements and Items

Several layouts are available to choose from. The number of elements is the number of cells in a layout. When you select a layout, the number of elements in the layout are automatically added for you to populate with your items. An item can be an image or video. When you add an item, there are additional properties you can define using the form that appears after clicking add item.

Items within an element are displayed sequentially on the slide and each item can have its own unique optional properties. Item duration values can be specified as a percentage (decimal value), in milliseconds, or in seconds: the script will convert whatever you put in this field to milliseconds. The total display time for a slide is determined by the duration specified in the top menu. When this time is reached, the slide will advance to the next slide in the index, even if the item currently being displayed has not reached the end of its display time: make sure your slide duration accounts for the sum of the item durations. Using a decimal lets you specify a percentage of the total duration as the item duration. Example: If you have four items, 0.25 for the item duration of each item will result in each item being displayed for 25% of the slide duration. 0.125 will result in each item being displayed twice, the items will be cycled through twice.

Some layouts have cells that are not all the same size and the content will need to have an aspect-ratio that matches the cell dimensions. Review the layout-specific instructions that will appear on the right-hand side of the screen when you select a layout so you know which element corresponds to which cell.

Item Properties

  • Content Type: This is where you specify if the item is an image or a video.
  • Content: This is where you specify the filename of your image or video.
  • Item Duration: This is where you specify the display duration for an item in a decimal, seconds, or milliseconds.
  • QR ID: Specifies the corner to display a QR code.
  • QR FIle: These are the QR code images available: default images redirect to the CVAD homepage.
  • Alt ID: Specifies the corner to display an Alt image.
  • Alt File: These are the Alt images available to choose from..
  • Alt BG: If using an Alt image that is transparent, you may want to see a background color.
  • Title Class: This for the positioning of overlaid text and style properties.
  • Slide Title: Title of the overlaid text.
  • Description: Text content of the overlaid text.
